# tools

Put each CLI tool in its own directory.

Current tools:

- `tokkit/`: lightweight local-first usage ledger for AI coding tools
- `yao-scai-cli/`: AI-native disk space scanner and advisor

Suggested pattern:

- `tools/<tool-name>/README.md`
- `tools/<tool-name>/src/` or `tools/<tool-name>/bin/`
- `tools/<tool-name>/package.json` or equivalent project file

Current tools:

- `tokscai`: categorized local memory usage viewer.
